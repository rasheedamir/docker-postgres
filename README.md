# Docker Postgres

Adds the following extensions on top of the [official 9.4 image](https://registry.hub.docker.com/_/postgres/)

  * [postgis](http://postgis.net/)
  * [cstore_fdw](https://github.com/citusdata/cstore_fdw)
  * [temporal_tables](http://pgxn.org/dist/temporal_tables/1.0.0/)
  * [mongo_fdw](https://github.com/citusdata/mongo_fdw)
  * [hashtypes](http://pgxn.org/dist/hashtypes/)
